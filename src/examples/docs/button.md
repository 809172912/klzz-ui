# Button 按钮
----
### 基础用法
使用```text```属性来定义 Button 的文本。

<div class="demo-block">
    <div>
        <div><kat-button></kat-button></div>
        <br>
        <div><kat-button text="hello world"></kat-button></div>
    </div>
</div>

::: demo

### Attributes
| 参数      | 说明      | 类型      | 可选值    | 默认值     |
|:--------- |:--------- |:--------- |:--------- |:---------  |
| text      | 文本      | string    | --        | kat-ui按钮 |
